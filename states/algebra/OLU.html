Линейные уравнения

Что такое линейное уравнение?

Линейное уравнение — это уравнение вида:
ax + b = 0 где:
x - неизвестная переменная,
a и b - числа (коэффициенты), причём a ≠ 0.
Примеры:
2x + 5 = 0
3x - 7 = 2
4(x + 1) = 12
Как решать линейные уравнения?
3 главных правила:
Можно переносить слагаемые через "=",меняя знак
Пример: x + 3 = 5 → x = 5 - 3 Можно умножать/делить обе части на одно и то же число,
Пример: 2x = 6 → x = 6/2
При раскрытии скобок знаки меняются, если перед скобкой минус.
Пример: -(2x - 3) = -2x + 3 Частые ошибки:
Забывают менять знак при переносе.
Неправильно: x + 5 = 3 → x = 3 + 5 Ошибки при раскрытии скобок
Неправильно: -(x - 2) = -x - 2 Путают, когда можно сокращать.
Нельзя: (2 + x)/x = 3/x → сокращать на x
